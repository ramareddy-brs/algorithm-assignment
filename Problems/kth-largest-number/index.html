Kth Largest Number

Implement an algorithm to find the "Kth" largest element in an array

This code is an implementation of a simple sorting algorithm called Bubble Sort. The purpose of the code is to sort the elements in the array arr in descending order. The outer loop iterates over the elements of the array, and the inner loop compares adjacent elements and swaps them if they are in the wrong order, i.e., if the current element is smaller than the next element
